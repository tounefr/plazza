Bienvenue dans la GALERIE ATARI :

Il suffit de cliquez sur les selections dans la colonne du haut pour naviguer sur :

    • DEMOS : les meilleurs démos sur STF/E.
      (certaines sont       disponibles en téléchargement...)
    • DENTROS : quelques compiles et minis démos sur ST.
    • MAGS : les principaux magazines français ATARI.
    • TRACKERS : tous les meilleurs trackers sur STF/E.
    • PLAYERS : les players pour écouter des modules sur STF/E.
    • UTILS : une overdose d'utilitaires pour Atari STF/E.
    • DIVERS : à ouvert ses portes le 15/03/2005 avec 8 compiles d'utilitaires.

Les démos disponibles sur le site fonctionnent aussi sur PACIFIST (version 0.49beta) WINSTON v 0.5 et STEEM v 3.2.
Les captures d'images sont effectuées sur les émulateurs ou rippées sur ATARI ST.
Les fichiers sont compressés avec Winzip v 8.0 avec une compression Maximum et peuvent être décompressés avec STZip (version 2.6) qui se trouve dans la section UTILS par exemple.

Tous les utilitaires PC/ATARI/AMIGA sont disponibles sur le serveur FTP
funet ou celui du serveur FTP de l'ibp.
L'émulateur PACIFIST et les utilitaires disques sont disponibles dans le fichier ZIP (546 Ko) ou sur le site de Frédéric Gidouin dans la section Download.

J'attire l'attention sur les personnes qui téléchargent les fichiers sur mon site et qui s'en servent sur leurs sites, comme par exemple des images de disks. Afin de récompenser le travail de dure labeur que j'ai effectué en vous mettant en ligne ses précieux fichiers
(transfert ATARI<->PC), j'espère que vous y glisserait ce petit logo ainsi que lien de mon site dessus. Si cela peut contribuer à de florissantes colaborations...


Attention : Ce site contient beaucoup d'images qui peuvent ralentir le chargement des pages...



Important : Les programmes et images de disks sont testés sur les émulateurs PaCifiST et Steem ainsi que sur des Atari 1040 STF/E et controlés sans virus connus !!!
