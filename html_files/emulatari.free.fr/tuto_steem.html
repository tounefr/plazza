[ Installation de Steem]
Tutoriel pour installer Steem sur un système Windows et le configurer afin de lancer des jeux Atari.
 

Bonjour,

Ce tutoriel a pour but d' installer le nécessaire sur votre système Windows pour émuler un Atari ST à l'aide de Steem. Et enfin il vous permettra de pouvoir jouer a des jeux Atari ST, à Buggy boy dans notre exemple.

Commençons par télécharger les fichiers nécessaires pour pouvoir commencer, à savoir:

- Une version récente de Steem, l'émulateur Atari ST pour Windows. Vous pouvez télécharger cette version en cliquant ICI
- Un fichier de traduction permettant d'avoir Steem en français, disponible ICI
- Une version du TOS, je vous conseille le TOS 1.02 fr pour utiliser des jeux, disponible ICI
- Une image disque d'un jeu à émuler : par exemple Buggy Boy, disponible ICI

Vous disposez donc maintenant de quatre fichiers sur votre disque dur :

  - Steem_v2_4.zip [ l'émulateur Atari STE pour Windows ]  
  - Translate_français.zip [ le fichier qui permet de franciser l'émulateur ]  
  - To102fr.zip [ l'image du systeme d'exploitation de l'Atari ST ]  
  - Buggy.zip [l'image de la disquette du jeu Buggy Boy ]  

On va pouvoir commencer l'installation.

D'abord décompressez le fichier steem_v2_4.zip dans un répertoire de votre choix, par exemple dans " c:\program files\steem ". Vous allez également décompresser le fichier tos102fr.zip afin d'obtenir le fichier tos102fr.img que vous allez placer dans le même répertoire qu'au dessus. Pour finir, décompressez aussi le fichier translate_français.zip et placez le fichier " translate_français.txt " toujours dans le même répertoire.

Félicitation, vous émulez un Atari ST. Pour utiliser la souris de l'Atari ST dans Steem, appuyez sur la touche Pause de votre clavier, pour récupérer la souris du PC ré appuyez sur la touche pause… simple non ?

Bon maintenant,voyons comment lancer un jeu. Fermez Steem. L'installation de Steem à directement lié les fichiers images qui contiennent les jeux ( .st, .msa, .dim, .stt ) à Steem. Ce qui veut dire que les fichiers images de jeux Atari ST s'executeront automatiquement sous steem si vous double cliquez dessus.

Décompressez votre fichier Buggy.zip, vous obtenez un fichier buggy.st qui doit ressembler à ça :

Double cliquez sur ce fichier et Steem se lancera automatiquement comme un Atari ST qui booterait sur cette disquette. Vous obtenez donc l'application Steem qui affiche le menu du jeu " Buggy Boy " :


Il reste un problème, vous n'avez pas de joystick pour jouer ! Il vous suffit d'appuyer une fois sur la touche " arret défil " du clavier de votre PC et vous pourrez émuler un joystick en utilisant les quatres flèches de direction et la touche " ctrl " pour faire feu.
