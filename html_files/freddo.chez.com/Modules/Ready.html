Quelques conseils d'utilisation de Mod Plugin :
Télécharger le logiciel sur le site officiel, puis installé le sur votre browser habituel si vous ne l'avez pas encore fait.
Sur l'applet : appuyer sur avec la souris pour écouter le module et appuyer sur pour stopper l'écoute (simple non !).
Pour avoir accès aux propriétées d'écoute : cliquez sur l'applet avec le bouton droit de la souris.
La version supérieur à 1.86 est conseillée pour écouter des modules S3M compressés avec WINZIP.
Equipements : un Pentium 166, 32 Mo de RAM (conseillé !) et de la patience car tout ce qui est Java fait ramé le système.
Si vous avez des problèmes ou pour plus de précision regarder le fichier FAQ de Mod Plugin.
